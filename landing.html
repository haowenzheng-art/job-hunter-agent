常见问题

关于数据、隐私、能力的核心疑问

JD 数据来源是什么？

公开招聘平台（JobsDB / 猎聘 / Liepin / Boss / 51job）公开 JD 爬取 + 用户手动添加。已对联系方式、人名等敏感信息脱敏，仅保留职位描述、要求、技能等结构化字段用于 RAG 检索。

我的简历数据安全吗？

简历以用户隔离方式存储在本地数据库（SQLite）或你自建的 PostgreSQL。系统不会上传简历到任何第三方服务，LLM 调用仅传递改写所需的最小上下文，不留存。

支持哪些岗位方向？

互联网、AI、产品、数据、设计、运营、工程等 30+ 岗位类目。内置 taxonomy 覆盖 AI 产品经理、LLM 工程师、数据科学家、RAG 架构师等新兴岗位，可基于 JD 库持续扩展。

LLM 用的是哪个模型？

默认接入 Agnes（apihub.agnes-ai.com），可在 .env 切换为火山引擎、OpenAI、DeepSeek 等 OpenAI 兼容接口，代码 provider-neutral，切换只改 4 个环境变量。

能导出 PDF 吗？

支持 Markdown 和 HTML 两种格式直接导出。PDF 可通过浏览器"打印 → 另存为 PDF"获得，排版已针对 A4 优化。

需要付费吗？

当前完全开源免费。LLM 调用费用由你自己的 API Key 承担，单次简历生成约 0.05-0.2 元。未来不排除推出托管 SaaS 版本，但开源版本永久免费。
